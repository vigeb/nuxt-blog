> Có thể dùng git cơ bản trên máy cũng như trên github

## Biết dùng git

Biết sử dụng những git command cơ bản như:

- git status
- git add -A,...
- git branch -d (branch name)
- git branch (branch name)
- git merge (branch name)
- git checkout (branch name)
- git commit -m "message"
- ...

## Biết sử dụng github

Biết cách push và pull cũng như clone project từ github bằng command

Biết cách merge branch trên trang web github.com
